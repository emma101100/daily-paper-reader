## 摘要
LLM智能体的长期记忆不仅仅是适时检索恰当段落。现有记忆系统将信念修正、因果耦合与跨领域抽象压缩到单一检索层面，仅针对表层回忆进行优化，因而在需要对用户演变进行推理的隐性个性化方面表现挣扎。我们提出DCPM，它按照认知能力层级重组智能体记忆，从原始输入和原子事实，经由历时信念轨迹与身份，上升到领域图式、潜在意图与跨领域模式。该层级由继承双过程理论架构分裂的两个过程驱动：同步的白昼写入器（系统1）将信念修正记录为双向链接的取代链，异步的夜间引擎（系统2）则归纳图式与意图，并扫描跨领域冲突，抽象为更高层次的核心图式。在LongMemEval、PersonaMem和PersonaMem-v2上，启用系统2在奖励隐性跨会话推理的基准上贡献最大（PersonaMem-v2最高提升+5.20），在跨度回忆上贡献最小，与架构预测一致。

## Abstract
Long-term memory for an LLM agent is more than retrieving the right passage at the right time. Current memory systems collapse belief revision, causal coupling, and cross-domain abstraction into a single retrieval surface tuned for surface recall, and consequently struggle on implicit personalisation that requires reasoning over how a user has evolved. We propose DCPM, which reorganises agent memory along a cognitive capability hierarchy ascending from raw inputs and atomic facts, through diachronic belief trajectories and identity, to domain schemas, latent intentions and cross-domain patterns. The hierarchy is driven by two processes inheriting the architectural split of dual-process theory: a synchronous daytime writer (System1) that records belief revisions as doubly linked supersedes chains, and an asynchronous nighttime engine (System2) that induces schemas and intentions and sweeps for cross-domain collisions abstracted into higher-level core schemas. On LongMemEval, PersonaMem and PersonaMem-v2, enabling System2 contributes most where the benchmark rewards implicit cross-session inference (up to +5.20 on PersonaMem-v2) and least on span recall, matching the architectural prediction.

## 1. 论文的核心问题与整体含义
- **研究动机**：现有LLM智能体的长期记忆系统大多将记忆视为“检索”问题，即通过向量存储、知识图谱等手段在查询时召回相关文本块。这类系统将信念修正、因果耦合和跨领域抽象压缩到单一检索层面，只能服务于表层事实回忆，无法有效支持需要推理**用户如何演变**的隐性个性化（如偏好迁移、意图变化）。因此，一个能理解用户深层认知演化的智能体需要超越单纯回忆。
- **整体含义**：论文提出将智能体记忆提升到“认知”层面，通过模仿人类认知架构（双过程理论）和能力发展阶梯，构建一个可以自演化的记忆系统。该系统能够追踪信念如何修正、发现行为背后的跨领域模式，并主动生成意图，实现从**被动存储**到**主动认知进化**的转变。

## 2. 论文提出的方法论
### 2.1 核心思想：认知能力层次
DCPM将记忆按认知能力组织为自下而上的层次：
- **基础层**：原始输入、原子事实、身份项目。
- **结构化层**：以双向链接的` SUPERSEDES / SUPERSEDED_BY `指针记录信念修正的连表（取代链），形成历时演化轨迹。
- **认知进化层**：在图谱中建立**领域图式**（总结行为规律）、**潜在意图**（对未来事件的关注）和**跨领域核心图式**（抽取出不同领域中行为相似但语义相异的深层模式）。

### 2.2 关键技术细节：双过程实现
- **系统1（白昼写入器）**：同步、按需触发（当用户或应用发出`add_memory`请求时）。
  - 步骤：①原始保存（零丢失）；②LLM提取身份项和原子事实；③LLM调解（对新旧事实执行`ADD`、`SUPERSEDE`、`UPDATE`或丢弃）；④批量更新向量库并建立取代链。
  - **取代链**是核心创新：不删除旧事实，而是以双向指针连接新旧版本，读取时通过一次命中即可遍历整个修订历史，恢复因果中间态。
- **系统2（夜间引擎）**：异步、宽松延迟，每天或闲时运行，分三阶段：
  - **阶段1**（无LLM预处理）：对未关联图式的新事实进行二阶段DBSCAN聚类（松→紧），获取与现有图式/意图节点的最近邻和反向证据边。
  - **阶段2**（LLM归纳）：为每个聚类调用归纳智能体，使用四个工具（`CREATE_SCHEMA`、`CREATE_INTENTION`、`ADD_EVIDENCE`、`ADD_EDGE`）增量构建图谱。
  - **阶段3**（跨域清扫器）：
    - a) 对每个图式生成行为描述，嵌入到行为认知空间E_beh（bge-m3编码），同时保留语义嵌入E_sem。
    - b) 扫描所有跨领域图式对，若满足：cos(E_beh) > θ_beh 且 cos(E_sem) < θ_sem，则标记为“冲突对”（行为相似但语义不同）。
    - c) 对冲突对聚类，调用LLM生成**核心图式**，并写入图谱。
- **读取路径**：无任何LLM调用，仅需向量最近邻搜索+取代链遍历+图谱证据边检索。

### 2.3 复杂度分析
- 写入：系统1为O(N log N)（HNSW查找）加少量LLM调用；系统2的预处理O(N)，归纳O(|C|)次LLM调用，清除O((D¯s)²)比较，其中领域数D≤20，图式数¯s≤30，在助手场景中可处理。
- 读取：O(log N + ¯k)，即单次近似最近邻查询加链遍历。

## 6. 主要结论与发现
- DCPM full在最具挑战的**隐含跨会话推理**任务（PersonaMem-v2）上取得显著提升，启用系统2带来+2.51（deepseek）至+5.20（kimi）的准确率增益，超过长上下文代理基线。
- **系统2的贡献高度不对称**：在表层回忆基准（LongMemEval、PersonaMem）上提升微弱（+0.33~2.41），而在需要跨域抽象推理的基准上增益最大，完全印证了双过程理论的预测。
- **取代链**是支持历时连续性的关键结构，约贡献2.9个点；跨域冲突检测生成的**核心图式**是认知层最主要的贡献者（约4.1个点），尽管其节点仅占总数1.3%。
- 该系统成功将记忆从“检索正确段落”提升为“对用户认知演化的建模”，读路径仍保持无LLM调用的低延迟。

## 7. 优点
- **认知导向的设计**：将记忆按认知能力分层的想法新颖，将心理学理论（双过程、心智理论）直接映射到工程架构。
- **取代链机制**：以双向链保留完整的信念修订历史，不丢弃中间态，为因果推理提供了结构支撑，是简单向量搜索无法实现的。
- **跨域抽象引擎**：通过行为-语义双空间碰撞检测，自动发现表面无关的深层行为模式，生成高度凝练的核心图式，效率极高。
- **读写分离且轻量**：读路径完全不依赖LLM，保证了推理低延迟；写入时为异步后台处理，不阻塞交互。
- **实验可解释性强**：通过多项消融和分析，清晰地量化了每个认知层的增益，验证了理论假设。

## 8. 不足与局限
- **场景限制**：仅评估了英文聊天助手场景，未涉及跨语言、具身智能体或用户身份不确定的情形。
- **超参数敏感**：双空间冲突的阈值θ_beh、θ_sem需在验证集上调参，不具有自适应性。
- **成本与可扩展性**：系统2的离线计算成本随图式数量增长；虽然当前领域数不大，但在极高领域多样性下二次项核对会成为瓶颈。
- **表示方式的局限**：认知节点使用自然语言便于解读，但缺乏结构化槽位，无法支持类似偏好向量的代数运算。
