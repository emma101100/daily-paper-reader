## 摘要
基于大语言模型（LLM）的智能体越来越依赖记忆来在持续交互中从经验里学习。然而，将经验存储为独立、扁平的单元会导致大量冗余和检索冲突，因为相似的片段会重复重叠内容，而细微的场景差异可能使检索到的记忆提供相互矛盾的指导。为解决这一问题，我们引入了残差经验的概念，即新获得的经验通常是对现有知识的增量变化。我们提出DeltaMem框架，将经验记忆组织成两棵独立的残差树：一棵将目标条件化的任务经验存储为可重用技能，另一棵存储场景级的环境知识。每棵树使用根节点保存通用基础经验，并使用增量delta节点存储后续变化，从而让相关经验在不重复的情况下共享一个共同基础。在检索时，通过带失败惩罚的相似性扫描找到最佳匹配，并通过从根到匹配的链式组合重构完整经验。自主整合机制将高频路径提炼为新的根节点，使树能够从通用启发式自组织为专门变体。在多种交互环境中的实验表明，DeltaMem一致优于现有基线方法。为促进未来研究，我们在https://github.com/import-myself/DeltaMem发布了代码。

## Abstract
Large Language Model (LLM)-based agents increasingly rely on memory to learn from experiences over continual interactions. However, storing experiences as independent, flat units leads to substantial redundancy and retrieval conflicts, as similar episodes repeat overlapping content and subtle scene variations cause retrieved memories to offer contradictory guidance. To address this, we introduce residual experience, positing that newly acquired experience is often an incremental variation of existing knowledge. We propose DeltaMem, a framework that organizes experience memory into two independent residual trees, one storing goal-conditioned task experience as reusable skills and another for scene-level environment knowledge. Each tree uses a root node for generalized base experiences and incremental delta nodes for subsequent variations, allowing related experiences to share a common foundation without duplication. For retrieval, a failure-penalized similarity scan locates the best match, reconstructing the full experience via root-to-match chain composition. An autonomous consolidation mechanism distills high-frequency paths into new root nodes, enabling the trees to self-organize from general heuristics to specialized variants. Experiments across diverse interactive environments show that DeltaMem consistently outperforms existing baselines. To facilitate future research, we release the code at https://github.com/import-myself/DeltaMem.

## 2. 方法论：核心思想与关键技术细节

- **核心思想**：将智能体经验拆分为两个正交维度：
  - **任务策略**：目标条件化的可执行步骤（技能）。
  - **环境知识**：场景级陈述性事实（对象位置、交互规则等）。
  并对每条新经验仅存储相对于已有最相似路径的**增量差异（残差）**，而非完整轨迹。
- **双树架构**：
  - **Task-Tree**（任务树）：节点存储可复用技能，形式为`⟨α, π, ω⟩`，其中α为激活条件，π为执行步骤序列，ω为终止条件。激活条件α用于检索匹配。
  - **Env-Tree**（环境树）：节点存储环境事实，形式为`⟨α, κ⟩`，α为场景描述触发器，κ为陈述性内容。
  - 两棵树独立异步演化，仅在推理时融合检索到的链，形成无干扰的决策上下文。
- **残差树结构**：
  - 每个节点定义为`v = ⟨type, label, d(v), c(v), e_v, P⟩`，包含类型（ROOT/RESIDUAL）、结果标签（成功/失败）、深度、成功命中次数、密度嵌入与实际载荷（技能或知识）。
  - **根节点**：存储某场景类的完整通用基础经验。
  - **残差节点**：仅存储相对于父链的增量载荷`∆P`，如额外步骤、边界条件扩展或新发现的对象状态，避免重复共享内容。
- **写入决策**：根据检索相似度`s(v* )`与阈值`τ_base`比较，低于阈值则创建根节点（冷启动），否则附加为最佳匹配链的残差节点。
  - 挂载规则：若最佳匹配节点深度已达上限`D_max`，则作为其兄弟节点附加，防止链无限增长。
- **检索与链重构**：
  - **全局扫描+失败惩罚**：对所有节点计算调整后相似度`s(v)=cos(e_q, e_v) - ε·I(label_v=FAILURE)`，取最高分节点作为匹配，并仅当`s(v*) ≥ τ_base`时接受。失败惩罚将过去失败作为检索护栏。
  - **链组合**：从根到匹配节点的路径按顺序组合完整上下文`C = ⟨P_v0, ∆P_v1, …, ∆P_vmatch⟩`，合并任务链和环境链注入LLM。
- **经验提取协议**：每轮交互后，LLM根据节点类型（根/残差）和回合结果（成功/失败）将原始轨迹提炼为结构化节点：
  - 成功根：提取完整基础技能，泛化环境特定常量。
  - 失败根：记录尝试动作与环境响应的具体失败点。
  - 成功残差：仅提取父链未覆盖的最小增量步骤，若完美覆盖则写入`skip`。
  - 失败残差：记录相对于检索链的确切中断点和环境反馈。
  - 环境树提取类似，但侧重类别级模式或新观察到的事实。
- **自主记忆整合**：
  - 跟踪每个非根节点的成功命中次数`c(v)`，当某路径的`c(v) ≥ K_cons`时，触发LLM将根到该节点的整条链蒸馏为一个新的基础技能或知识，作为新根节点写回原树，原节点标记为已整合。
  - 使记忆层次从通用启发式逐步自组织为专门变体，压缩高频路径，降低树深度。

## 6. 主要结论与发现

- **整体性能**：DELTAMEM在三个基准的6个划分中均取得最佳或媲美最佳的结果，尤其在ALFWorld和ScienceWorld上显著超越所有基线。
- **冗余与冲突解决**：残差节点平均token数约为根节点的一半，证实共享内容未被重复存储，而检索链重构有效消除冲突。
- **记忆库复用**：预建残差树在冻结状态下迁移到新回合仍表现优异，解冻后继续更新可进一步提升，说明记忆可离线和在线混合使用；相比之下，AWM和ReasoningBank在线更新可能引入冲突修改导致退化。
- **双树互补**：消融实验表明，仅留任一单树性能均明显下降，验证任务树和环境树捕获了正交的经验维度。
- **参数鲁棒性与自组织**：在整合机制激活时，性能对检索阈值不敏感，表明整合具有正则化效果；随整合阈值`K_cons`降低，树深度减小，高频路径被压入根层，实现自组织。

## 7. 优点

- **新颖的残差存储范式**：将经验存储为相对于已有知识的增量残差，从根源上消除冗余，同时保持语义完整性。
- **双树解耦**：明确分离任务策略与环境知识，使不同维度独立检索和演化，避免跨任务干扰，提升泛化能力。
- **失败惩罚检索**：将失败记忆转为检索护栏，利用过去错误提升决策安全性，而非简单丢弃。
- **自主整合机制**：受神经科学记忆巩固启发，使记忆树随学习推进自动压缩高频路径，从扁平到分层自组织，无需外部路由缓存。
- **实验覆盖广**：横跨具身、科学、电商三大领域，与多种不同范式的基线对比，并包含离线和在线两种记忆使用模式。
- **开源代码**：提供完整实现，便于复现和未来研究。

## 8. 不足与局限

- **无时态衰减**：当前框架未建模经验的时间衰减，过时环境状态（如网站布局变更）可能持续存在并误导检索。
