## 摘要
智能体大语言模型必须持续决定新提取的事实是应该被添加、与现有记忆合并还是被忽略，然而以往的工作更侧重于检索和存储，而非原则性的写入端控制。我们将记忆演化框架化为新颖性检测问题，并提出SAGE，即一种基于球面自适应的记忆演化门控。它利用基于冯·米塞斯-费舍尔分布的密度估计器对记忆嵌入进行打分，并通过跟踪记忆存储几何的自适应阈值来路由候选事实。SAGE将明显新颖的事实判定为添加，将明显冗余的事实判定为无操作，仅将不确定的案例发送到LLM合并步骤，从而减少昂贵的写入时推理。在LoCoMo数据集上，与Mem0相比，SAGE在所有七个开放权重主干对比中取得了最佳的平均Token-F1分数，而在GPT-4o-mini上，它将添加阶段的API成本降低了3.4倍，添加阶段的延迟降低了2.5倍，且平均评判分数差距很小。作为A-Mem的即插即用二元门控，SAGE在五个模型上跳过了约16-18%的LLM调用，且在开放权重主干上质量变化极小。这些结果表明，新颖性感知的写入控制是提升长期智能体记忆中记忆质量和系统效率的实用杠杆。

## Abstract
Agentic LLMs must continuously decide whether newly extracted facts should be added, merged with existing memories, or ignored, yet prior work has focused more on retrieval and storage than on principled write-side control. We frame memory evolution as a novelty-detection problem and propose SAGE, a Spherical Adaptive Gate for memory Evolution that scores candidate facts with a von Mises-Fisher-based density estimator over memory embeddings and routes them with an adaptive threshold that tracks memory-store geometry. SAGE resolves clearly novel facts as ADD, clearly redundant facts as NOOP, and sends only uncertain cases to an LLM merge step, reducing expensive write-time reasoning. On LoCoMo, SAGE achieves the best average token-F1 against Mem0 on all seven open-weight backbone comparisons, while on GPT-4o-mini it reduces add-phase API cost by 3.4$\times$ and add-phase latency by 2.5$\times$ with only a small average judge-score gap. As a drop-in binary gate for A-Mem, SAGE skips roughly 16-18% of LLM calls across five models with minimal quality change on open-weight backbones. These results suggest that novelty-aware write control is a practical lever for improving both memory quality and system efficiency in long-term agentic memory.

### 1. 论文的核心问题与整体含义

*   **核心问题**：智能体大语言模型在长期交互中，需要持续决定从对话中提取的新事实是应该**添加（ADD）**、**更新/合并（UPDATE）** 到现有记忆，还是直接**忽略（NOOP）**。这个“写入端”决策至关重要，但以往研究（如Mem0, A-Mem）将过多精力投入在“检索”和“存储”上，对“写什么”缺乏有原则性的控制，通常将此决策完全委托给昂贵的LLM调用，导致写路径成为系统的主要算力瓶颈。
*   **研究动机与整体含义**：作者认为，记忆演化的本质是一个**新颖性检测**问题。即，在写入前应先判断一条候选事实相比现有记忆是否足够“新”。基于此，论文旨在设计一个廉价、闭环的计算门控，在LLM介入前过滤掉明显冗余或明显新颖的事实，仅将模糊案例留给LLM处理，从而在保证记忆质量的同时，大幅提升系统效率。整体而言，该工作将“写端控制”从一个被忽视的成本中心，转变为同时提升质量和效率的关键杠杆。

### 2. 论文提出的方法论

SAGE (球形自适应门控) 是一种基于冯·米塞斯-费舍尔密度估计的新颖性门控机制。

*   **核心思想**：利用核密度估计来评估候选事实嵌入在当前记忆库支持的密度，从而量化其新颖性。密度高（新颖性低）则忽略，密度低（新颖性高）则直接添加，密度处于中间模糊地带才触发LLM进行合并/更新操作。
*   **关键技术细节与算法流程**：
    1.  **嵌入与向量化**：将对话中提取的候选事实 `c` 和已存储的记忆项通过句子嵌入模型转换为向量，并归一化到单位超球面上。
    2.  **vMF新颖性打分 (svMF)**：基于冯·米塞斯-费舍尔分布，定义一个核函数 `K(c, m_i) = exp(κ * m_i^T * c)`，其中 `m_i` 是记忆库中的记忆，`κ` 是浓度参数。通过对所有记忆的核函数值取平均、对数化并缩放，得到一个新颖性分数 `ν(c)`。
        *   该分数不同于简单的余弦相似度，它不仅考虑与最近记忆的距离，还**综合考虑了该区域内所有记忆的支持密度**。分数越高，代表事实越新颖。
    3.  **自适应路由规则**：定义一个基于记忆库几何密度的自适应阈值 `τ_t`。记忆库越密集（记忆向量分布越集中），阈值越低，门控越宽松，以避免在密集区过度拒绝新事实。路由逻辑如下：
        *   **N = 0 (空记忆库)**：执行 `ADD`。
        *   **ν(c) ≥ τ_t + δ**：事实非常新颖，直接 `ADD`（不调用LLM）。
        *   **ν(c) < τ_t**：事实高度冗余，执行 `NOOP`（不调用LLM）。
        *   **τ_t ≤ ν(c) < τ_t + δ**：事实落入“不确定带”，触发LLM进行 `UPDATE`（合并等操作）。
    4.  **即插即用的NOOP门控**：SAGE还可以作为一个独立的二进制预过滤器，部署在任何现有记忆系统（如A-Mem）的上游。它使用一个固定的高阈值 `τ_noop`，仅当 `svMF` 分数很高时才判定为 `NOOP` 并丢弃该事实，否则将事实向下传递，由宿主系统自行处理，实现非侵入式升级。

### 7. 优点

*   **问题定义新颖且深刻**：将记忆演化重新定义为“新颖性检测”，为解决长期智能体记忆的写控制问题提供了全新且准确的理论视角。
*   **方法简洁高效**：SAGE方法基于成熟的vMF分布和核密度估计，计算开销小，无需训练，可闭环决策，实现了低成本和高效率的统一。
*   **设计精巧**：自适应阈值的设计巧妙地解决了记忆库从稀疏到密集时，固定阈值容易失效的问题，实现了与记忆几何的动态匹配。“不确定带”则优雅地平衡了自动化决策与LLM高级推理的成本和收益。
*   **实验全面且有说服力**：实验覆盖多个模型、两种应用模式（系统级和模块级），并从质量、延迟、成本、调用次数等多维度进行了“头对头”的公平对比，证据链完整。
*   **极强的实用性**：作为“即插即用”模块，SAGE可以无缝集成到现有记忆系统中，无需修改宿主系统，部署门槛低，实用性很强。

### 8. 不足与局限

*   **评估场景有限**：实验仅在LoCoMo这一个英语长对话基准上进行，未在更多样化的智能体任务（如工具使用、代码生成）或跨语言场景中验证泛化能力。
*   **功能不完整**：目前SAGE仅实现了ADD、NOOP和UPDATE的路由，**未包含明确的DELETE（删除）操作**和记忆压缩（Compaction）机制，这些对于长期运行的智能体来说同样重要。
*   **依赖底层嵌入模型**：SAGE的性能高度依赖于句子嵌入模型的质量。模型若将语义不同但向量相近的事实混淆，SAGE会错误丢弃；反之，对同义句产生差异较大向量，则可能绕过冗余过滤。
*   **部分指标不占优**：尽管整体效率和质量有优势，但在GPT-4o-mini的对比中，SAGE的单跳和时序问题的LLM评判分数（J-Score）明显低于Mem0，表明其在某些特定类型的问题上存在精确度损失。
