## 摘要
记忆增强型LLM智能体通过递归地将交互轨迹总结为紧凑的记忆，来解决复杂的长时间范围任务。然而，现有方法通常使用基于结果的强化学习来训练这些记忆策略，无法定位中间记忆质量退化的位置。随着交互的展开，模糊的递归总结逐步丢弃与任务相关的信息，并引入语义噪声。这加剧了信念偏差，模糊了智能体对潜在任务状态的估计，最终使长时间范围推理偏离轨道。因此，我们主张记忆优化不应仅关注轨迹级别的成功，而应关注中间总结所诱导信念的清晰度。为此，我们引入了信念熵（Belief Entropy），一种自监督代理，用于探测模型在给定当前记忆的情况下对潜在任务状态的不确定程度。基于这一代理，我们提出了元认知记忆策略优化（Metacognitive Memory Policy Optimization, MMPO）。MMPO不仅依赖稀疏的基于结果的信号，还通过显式惩罚诱导高认知不确定性的总结，提供细粒度、记忆特定的监督。实验表明，MMPO在多种长时间范围任务上持续优于现有方法，即使扩展到175万token上下文，仍保持97.1%的性能。

## Abstract
Memory-augmented LLM agents tackle complex long-horizon tasks by recursively summarizing interaction trajectories into compact memory. However, existing approaches typically train these memory policies using outcome-based reinforcement learning, failing to localize where intermediate memory quality degrades. As interactions unfold, ambiguous recursive summaries progressively discard task-relevant information and introduce semantic noise. This exacerbates belief deviation, obscuring the agent's estimate of the latent task state and ultimately derailing long-horizon reasoning. We therefore argue that memory optimization should focus not merely on trajectory-level success, but on the clarity of the belief induced by intermediate summaries. To this end, we introduce Belief Entropy, a self-supervised proxy that probes how uncertain the model remains about the latent task state given its current memory. Based on this proxy, we propose Metacognitive Memory Policy Optimization (MMPO). Instead of relying only on sparse outcome-based signals, MMPO provides fine-grained, memory-specific supervision via explicitly penalizing summaries that induce high epistemic uncertainty. Experiments show that MMPO consistently outperforms existing methods on diverse long-horizon tasks, maintaining 97.1% performance even when scaled to 1.75M-token contexts.

## 2. 论文提出的方法论

### 2.1 形式化框架与核心思想

- **POMDP建模**：  
  将长时间范围任务建模为部分可观测马尔可夫决策过程（POMDP）⟨S, A, Ω, T, O, R, γ⟩。  
  真实任务状态 *s* 不可直接观测，智能体根据交互历史 *h* 维持一个**信念状态** *b(s)* = P(*s*|*h*)。  
  在记忆压缩的设定下，信念由压缩后的摘要 *m* 诱导：  
  \[
  b^M(s) \triangleq P(s \mid m)
  \]
  理想情况下，记忆策略应使该摘要诱导信念尽可能精准，即**最小化条件熵 H(s|m)**，等价于最大化 *s* 与 *m* 之间的互信息。

- **信念熵（Belief Entropy）代理**：  
  由于潜在状态 *s* 不可观测，无法直接优化 H(s|m)。  
  作者受认知科学中“元认知探测”启发，设计一个**锚问题（anchor question）**（如：“基于当前记忆，当前任务进展如何？还需哪些信息？”），并定义信念熵为该问题的回答 *y* 在模型下的**预测熵**：
  \[
  H_{BE}(m) \triangleq H(y \mid m, q)
  \]
  通过计算贪心回答 *y*\* 上每个 token 的预测熵均值得到经验估计 \(\widehat{H}_{BE}(m)\)。  
  直觉上，若摘要清晰、包含必要信息，回答应当确定性强、熵低；反之，摘要模糊或缺失信息时，回答熵高。

- **信息论依据**：  
  利用链式法则 \(H(y|m,q) = H(y|m,q,s) + I(y;s|m,q)\)。  
  其中第二项捕捉了 **通过回答暴露出的潜在状态剩余不确定性**，因而信念熵能作为 H(s|m) 的一个有效代理（见附录 C）。  
  实证验证表明：成功轨迹信念熵逐步下降，失败轨迹则升高或停滞；熵的减少量与最终任务精度强相关；甚至在推理时直接选择低熵轨迹也能提高准确率。

### 2.2 MMPO 训练框架

MMPO 不改动记忆生成和行动策略的架构，**仅增强训练信号**，具体步骤如下（参见正文算法1和 Figure 4）：

#### 阶段1：轨迹采样
- 对每个任务，从记忆策略 \(\pi_\theta\) 中采样 **G** 条完整的交互轨迹 \(\{\tau^{(i)}\}\)。

#### 阶段2：子轨迹奖励计算
- 对每条轨迹的每个轮次 *k*，生成记忆摘要 \(m^{(i)}_k\)，并用式(5)计算信念熵 \(\widehat{H}_{BE}(m^{(i)}_k)\)。
- 构建**子轨迹奖励**（sub‑trajectory reward）：
  \[
  R^{(i)}_k = \alpha \cdot \sigma\!\left(-\widehat{H}_{BE}(m^{(i)}_k)\right) + r^{(i)}_{\text{final}}
  \]
  其中 \(\sigma\) 为 sigmoid 函数，将信念熵映射到 (0,1) 并与最终结果奖励 \(r_{\text{final}}\) 结合，既保持与最终成功挂钩，又在**每一步提供了密集的过程信号**。

#### 阶段3：优势估计与策略优化
- 采用**组相对优势估计**（类 GRPO）：对同一任务采样的一组轨迹，在相同深度 *k* 的子轨迹奖励上进行标准化：
  \[
  \widehat{A}^{(i)}_k = \frac{R^{(i)}_k - \text{mean}(R_k)}{\text{std}(R_k)}
  \]
- 为正确分配信用，将每个轮次 *t* 的摘要质量与**所有受其影响的后续子轨迹**关联，计算该轮次的优势：
  \[
  A^{(i)}_t = \frac{1}{T-t+1} \sum_{k=t}^{T} \widehat{A}^{(i)}_k
  \]
- 使用**带裁剪的 PPO 目标**更新记忆策略 \(\pi_\theta\)，并加入对参考模型的 KL 散度惩罚项，防止崩溃。

这一设计使 MMPO 在不改变递归记忆工作流的前提下，仅靠“记忆状态上的元认知密集奖励”显著改善信用分配，是方法论的核心创新。

## 6. 论文的主要结论与发现

1. **信念熵是有效的记忆质量代理**：  
   - 成功轨迹信念熵逐渐下降，失败轨迹升高；  
   - 熵的下降量与最终任务精度呈强负相关（r ≈ ‑0.68）；  
   - 推理时仅通过选择低熵轨迹即可提升准确率。

2. **MMPO 显著改善长时间范围推理**：  
   - 在 RULER‑HotpotQA 上，MMPO‑7B/14B 在绝大部分上下文长度下均优于 RL‑MemAgent，尤其在 224K～3.5M 超长上下文下平均提升 +3.1%～+3.2% 准确率，最大提升达 +5.47%（7B 模型，896K 长度）。  
   - 即使上下文长达 1.75M tokens，MMPO 仍能保持 97.1%（相对基准）的性能。

3. **方法具备跨任务、跨框架的泛化性**：  
   - 在 MEM1 的多目标 QA 中，16 目标难度下 MMPO 比 MEM1‑QA 的 F1 提高 0.45 以上；  
   - 在 WebShop 交互任务中，MMPO 将平均奖励由 70.87 提升至 77.25。

4. **锚问题设计至关重要**：  
   - “进展+缺失信息”的联合探针优于单纯询问答案或仅问缺失信息，能更准确地暴露任务状态的不确定性。

这些发现表明，将元认知不确定性与结果奖励相结合，能够有效解决长期记忆策略优化中的信用分配难题，为构建稳定长时程智能体提供了实用原则。

## 7. 优点

- **理论坚实且直觉清晰**：  
  从 POMDP 信念偏差出发，将记忆优化归结为“最小化条件熵 H(s|m)”，再用认知科学中的元认知探测将其转换为可操作的信念熵，整个推导链严密，易于理解。

- **自监督密集过程监督**：  
  信念熵完全来自模型自身对锚问题的回答分布，无需额外标注或环境奖励，可在任意文本交互场景中低开销地产生中间监督信号。

- **方法轻量且可移植**：  
  MMPO 不改变记忆生成、交互架构，仅增补训练奖励，可直接“插入” MemAgent、MEM1 等现有框架，展现出良好的兼容性和增益。

- **全面的实验验证**：  
  覆盖多任务、多框架、多长度、消融与信号选择分析，证据充分；同时提供推理时熵选优的单独实验，进一步强化了信念熵信号的内在价值。

- **超越简单置信度的信号设计**：  
  对比实验显示，单纯直接回答熵或记忆长度等简单代理效果远不如信念熵，说明设计锚问题以暴露“缺失信息”而非“已有答案”，是至关重要的。

## 8. 不足与局限

- **锚问题的设计依赖性**：  
  信念熵的有效性高度依赖锚问题的质量，当前模板需针对任务类型手动设计，对全新的开放任务可能缺乏通用性；文中也承认需要更自适应的探测方法。

- **不能直接度量真实状态不确定性**：  
  信念熵只是 H(s|m) 的代理，通过回答分布间接反映。若模型本身对锚问题生成十分确定但与状态无关（如虚构进展），则信号可能被误导，目前仅通过结果奖励约束。

- **实验局限于中等规模的 Qwen 模型**：  
  仅测试了 7B 和 14B 参数模型，未验证在更大模型、更强基础能力上的表现；同时评估任务仍以知识密集型和检索型为主，对复杂工具调用、多轮多步交互的覆盖度有待提升。

- **计算开销与实现细节未完全公开**：  
  虽然给出了相对开销比例，但未报告 GPU 型号、训练时间等具体算力资源，影响完全复现性。

- **可能的风险**：  
  过度优化信念熵可能导致记忆描写过于“确定”，但忽略了一些合理的不确定性表达；在高度不确定或开放环境中，如何平衡熵奖励与任务奖励仍需研究。

总结而言，MMPO 为长时间范围智能体提供了一种优雅、理论驱动的记忆监督范式，其核心贡献在于将“记忆引发的信念清晰度”量化为可优化的信号，在学术和应用层面均有重要价值。后续若能在更广泛的任务、模型规模和自适应锚问题设计上进一步拓展，有望成为长时程记忆优化的基础方法。
